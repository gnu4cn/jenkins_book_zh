# Jenkins 全知道

有关 Jenkins CI/CD 的全面资料。

Jenkins 是一款由 Java 编写的开源的持续集成工具。Jenkins 的扩展插件已经发布，能使非 Java 语言编写的项目也使用 Jenkins。对于大多数的版本控制系统和大的数据库，有与 Jenkins 集成的插件可用。许多构建（build）工具都是通过他们各自的插件提供支持。插件还可以改变 Jenkins 的外观，或添加新的功能。
